## v0.1（开发中，含 M1–M6；起 2026-06-19）

> v0.1 是涵盖 6 个里程碑的整体内容版本，逐个里程碑推进、不单独发版；M1 处打过一个 `v0.1` 检查点标签，最终发版时再定标签指向。

### 类肉鸽无尽模式（大改版）
- **删除购买系统**：金币/右侧基础塔面板/拆除返金全部移除。开局点「开始」弹**三选一**挑初始猪塔（免费），点塔位放置。
- **无尽刷分取代关卡制**：击杀得分数+经验；**分数=难度曲线**——出怪间隔随分缩短、狼血随分增厚(每600分+100%)、移速缓涨；分数解锁新怪：250 疾风狼、700 铁背狼、**1500 头狼(Boss)**、**3500 羊皮狼(Boss)**，Boss 解锁后每 1200 分再来一只。国王阵亡=结算（最终分/历史最佳，最佳分持久化）。
- **升级模式改版（弃拖拽合成）**：经验条升级→弹**三选一（激光L/元素E/实弹B）**→点一座塔升级（塔+元素=确定产物，见 tower_defs.UPGRADE 表：一级+元素=二级；二级+元素=对应三级；三级封顶）或点空塔位放该元素新塔。等级越高升级越慢(经验需求×1.22/级)。**满级=15次行动**(5塔各升满三级)，之后升级改为**属性三选一**：攻击+10%/攻速+8%/暴击+5%(2倍伤害)，可无限叠。
- **塔上限 5 座**。弹窗期间战场冻结；升级表 39 项断言 0 失败。
- 验证：无头自动战——1塔起步→8级→3塔→1107分(新怪如期解锁)→被狼潮淹没出结算，全程零报错。

### 割草模式（车道三版）：全屏索敌 + 1格塔 + 怪量翻倍
- **索敌改全屏锁定**：怪一出场（含越线围攻国王的）塔就能打，无任何范围/车道/距离限制；目标优先级=离国王最近者。激光恢复瞄准跟随的方向光束（LL 双束）、轨道炮朝最高威胁贯穿、领域恢复塔周身圆形、布雷全场随机撒雷。
- **塔占地 1 格**：一排最多约 28 座塔（原 2×2 改 1×1，塔身缩小适配）；塔位带改为一格高的一排。
- **怪量约 ×2.5、出怪间隔约减半**（割草弹幕密度）：L1 第一波 15 只起步、后期波 40+；L2 同步上调。
- **国王血条移到国王右侧**（原头顶位置会顶进塔位带）。
- 验证：22 座 1 格塔全屏索敌硬扛 L1 全 5 波（约 90 只狼），满血通关、伪装/狂暴断言通过、零报错。

### 车道模式二版：国王后置 + 只向前攻击 + 伤害数字
- **布局调整**：塔位一整排（国王移出塔位带），**国王在最底部中央的王座区**；狼越防线后穿过塔排冲向国王。
- **取消攻击范围**：删掉范围圈。所有塔**只朝正前方（上方）自己那条竖道攻击**，无距离上限（打到战场顶部）；塔身自带淡色竖道提示攻击区。
  - 激光/轨道炮改为垂直向上贯穿本车道；领域塔改为塔前竖向元素地带；布雷车在本车道内撒雷；狙击/标记/弹道均在本车道内索敌。
  - 例外保持区域性质：双子圆环、等离子塔间连线、光环增益、棱镜分裂的次级激光（主目标仍在本车道）。
  - **注意**：漏过防线的狼塔打不到（严格只向前），只有圆环/等离子线/爆炸波及可清——漏怪代价显著变大。
- **伤害数字**：敌人受伤飘出伤害数字（0.4s 合并一次防止激光刷屏；大额伤害更大更黄；击杀立即结算）。`scripts/damage_number.gd`。
- 验证：无头自动战零报错；本版布阵下 5 波败北（漏怪围攻国王致死）——胜负两路径均验证过，难度留实测调。

### 玩法结构改版：竖版车道模式（首版）
- **场地重构**：上方整片为狼群冲锋区——狼从**顶端随机横坐标出现、垂直往下走**；下方一条**塔位带**（两格高，正好一排 2×2 塔）；塔位带上缘为红色**防线**，狼越线即转为围攻国王。
- 国王移到塔位带中央（560,600），周围保留禁建；右侧 UI 栏（面板/作弊器）不变。
- 关卡数据简化：不再有路径路点/岔路，两关改为「草原防线 / 雪原防线」（配色区分，波次沿用）。
- 适配：布雷车改为在塔上方冲锋区随机撒雷；轨道炮"走得最远"改为按剩余路程判定；其余塔行为/合成/部署/作弊全部沿用。
- 验证：无头自动战——13 塔沿防线布置、狼压线围攻国王（掉 96 血后被清场）、5 波胜利、伪装/狂暴断言通过、零报错。

### M3 修正（头狼技能改版 + UI 布局分区）
- **头狼技能改版**：废弃"改路"（原实现会脱离画出的路径）。新技能：血量每掉过 **75%/50%/25%** → **瞬间狂暴加速**（2.4 倍速、无视减速，持续 1.6s，身体泛红）并**清除自身所有负面效果**（减速/灼烧/标记）。
- **建造区分区**：明确划定建造范围——**格子中心 x>1120 的右侧 UI 栏永久禁建**；基础塔面板与**作弊器面板都移到右栏**（作弊器在面板下方，悬停名称改左弹）。国王周围禁建保留。第二关国王移位（1170→1130），不再与基础塔 UI 重叠（第一关国王同步移位）。
- 移除备用岔路的地图绘制与注入（技能已改）。
- 验证：狂暴断言（减速39→狂暴187.2、负面清除）、全程自动战通关零报错。

### M3（双 Boss 机制 + 第二关 + 通关解锁）
- **头狼（Boss①·改路）**：血量每掉过 75%/50%/25% 阈值，跳到本关**备用岔路**继续进军（有红圈嚎叫提示）；岔路在地图上以淡色路面常显。
- **披着羊皮的狼（Boss②·伪装）**：登场即**伪装态**（羊皮白半透明）——**成分不含 E 的塔完全锁不住它**（弹道也打不中）；只有**元素(E)系**的塔/弹药/地雷/圆环能识破并命中。出场带机制提示。
- **第二关「黑森林雪原」接入**：通关第一关解锁；胜利后按 **N** 进下一关，按 **1/2** 选关（未解锁会提示）；HUD 显示当前关名。
- **进度持久化**：解锁进度存 `user://save.json`，跨局保留。
- 修复：光束爆裂炮降伤(70→34)、引爆时间 3s→1s（上一提交）。
- 验证：无头断言——伪装狼 `hittable_by(B)=false / (E)=true`；头狼掉血后路点切换；两关都自动战通关（第二关 E 系塔击杀伪装 Boss），胜利正确解锁下一关，零报错。

### 作弊器修复
- **修复作弊拖放崩溃**（"进入脚本页面"）：根因是 `_finish_deploy` 无条件取 `def["cost"]`，而高级塔定义无 `cost` 字段→运行时报错。改为 `.get("cost", 0)`。
- **作弊塔悬停显示名称**：鼠标移到左侧作弊网格的塔上，旁边显示该塔名称（格子也高亮）。
- HUD 文字标签设为不拦截鼠标（`MOUSE_FILTER_IGNORE`），避免中央提示标签遮挡地图点击。

### 修复批次（爆裂炮/双子/面板/作弊/占地）
- **光束爆裂炮**：锁定单一目标持续照射（不再被最近怪打断），满 3 秒**爆炸有可视化光环**；**激光阶段不再附元素**，只有爆炸那一下才附（几率）。
- **双子激光塔**：圆环范围放大到**与攻击距离相等**。
- **面板可建选项实时亮起**：修复"金币够了但不动鼠标选项还是灰的"——根因是面板画在 `_draw` 里、金币变化未触发重绘；改为 `GameState.changed` 时主动 `queue_redraw`。
- **开发版作弊**：按 **C** 开关，左侧出现全 30 塔网格，拖出即**免费直接放置任意高级塔**（无需合成、无需金币），方便测试。
- **占地统一为 4 格**：所有塔统一 2×2。

### 部署即合成 + 6 个塔攻击方式定制
- **面板拖拽可直接合成**：从右侧面板把基础塔拖到**已有同级塔**上松手即合成（不止能放空地）；拖到塔上显示绿/红合成提示。
- **聚焦激光(LL)**：同时发射**两束激光打两个敌人**。
- **机枪塔(BB)**：**双管并排两颗子弹**，基础攻速略慢、越打越快（上限 0.1s 间隔），**停火 3 秒重置**攻速。
- **光束爆裂炮**：激光持续照射同一目标**满 3 秒触发范围爆炸**，爆炸对周围敌人有几率附带元素效果。
- **元素矩阵**：改为**同时朝最多 3 名敌人各射一道弹道激光**，命中后向最近敌人**弹射 2 次**并附元素效果（`Projectile` 新增 bounce 模式）。
- **双子激光塔**：改为**元素圆环扩散→收缩**，环边碰到的敌人受伤（扩散/收缩各一次共两段），**连发 2 个环后进 2 秒小 cd**，每环元素随机、有几率附状态（`scripts/ring.gd`）。
- 验证：26 种各原型塔全跑通，5 波零报错。

### 建造交互改版 + 占地统一 + 轨道炮发射按钮
- **右侧面板拖拽部署**：屏幕右侧常驻三个基础塔（激光猪/元素猪/机枪猪），**从面板拖到地图**松手部署；拖动过程实时显示**占地格**（绿可放/红不可或钱不够）。取消了"点空地弹三选一"的旧流程。
- **悬停显示塔名**：鼠标移到已放置的塔上，上方显示该塔名称。
- **占地统一**：一级塔 1 格、二级塔 4 格（2×2）、三级塔 6 格（3×2）。
- **轨道炮发射**：改为塔**上方弹出「发射」按钮**（带充能百分比），点按钮发射；**未蓄满也能手动射、伤害按充能比例打折**；蓄满有目标仍自动满威力发射。取消了"轻点塔身发射"。
- 验证：22 种各原型塔 + L+L→LL 合成，5 波零报错。

### 防御塔调整（等离子/布雷/棱镜/轨道炮）
- **等离子场**：不再只连同类，**连接范围内所有类型的塔**，敌人穿过连线受伤并被**减速**。
- **自动布雷车**：地雷**伤害下调**（55→30）、**爆炸范围缩小**（100→64）。
- **棱镜塔**：实现真·折射分裂——主激光命中→分裂最多 3 道小激光打附近敌人→每道再分裂最多 3 道小小激光（两级分裂，伤害逐级衰减），并**整体降伤**（60→26）；分裂光束有可视化连线。
- **轨道炮**：加**充能进度条**（蓄满变绿），蓄满显示"蓄满! 点击发射"提示；支持**点击塔手动发射**（轻点不拖走即触发），蓄满有目标时也会自动发射。
- 验证：22 种各原型塔（含棱镜分裂/等离子连线/轨道炮）全跑通，5 波零报错。

### 手感修复二（提示/金币/结束/弹道）
- **合成等临时提示自动消失**：合成/拆除/金币不足/放不下/不可合成等提示 2 秒后自动清空（开始/准备/胜负提示仍常驻）。
- **建造弹窗金币实时刷新**：弹窗开着时，攒够钱的选项会**自动点亮**，不用关掉重开。
- **游戏结束全场冻结**：失败/胜利后敌人、塔、子弹、地雷、特效全部停止（之前结束后敌人还在动）；刷怪与波次计时也停。
- **弹道塔忠实化**：
  - 爆裂榴弹炮：爆炸后真的分裂出 **3 颗可见小炸弹** 二次爆炸；所有范围爆炸加了**爆炸光环视觉**（不再只看到敌人掉血）。
  - 弹幕炮塔：改为**扇形齐射 12 发**元素弹、每发随机元素、攻速调慢（不再单发弹道）；元素风暴=3 球、元素榴弹阵列=6 发齐射同此机制。
- 新增 `scripts/boom.gd`（爆炸视觉）；验证：22 种各原型塔含齐射/分裂全跑通，5 波零报错。

### 手感修复（拖拽移动 + 攻击原型还原）
- **塔可在战斗中移动**：长按塔→跟随鼠标→触碰的可放置格子亮起（绿可放/红不可）→松手放置；落到同级塔上则合成。**去掉了拖拽指引线**。放不下自动退回原位。
- **攻击原型严格按设计稿还原**（不再一律弹道）：新增 `tower_defs` 的 `ATTACK` 原型表 + 重写 `tower.gd` 分派——
  - **beam 持续激光**：一道直线激光瞄准跟随敌人，沿线持续 DPS、可穿透 N 个（激光猪/聚焦激光/元素激光/双子/棱镜/光束爆裂等）。
  - **mine 布雷**：自动布雷车无论有无敌人都按频率往附近路径埋地雷，敌人踩中爆炸（`scripts/mine.gd`）。
  - **domain 领域**（元素反应炉周身持续 AoE）、**charge 充能炮**（轨道炮蓄满一道贯穿长线）、**snipe 狙击**（反器材/精准狙击锁血最高、瞬发巨伤）、**link 塔间连线**（等离子场）、**aura 光环增益**（催化反应塔加成周围塔）、**mark 集火标记**（战术标记炮放大全体伤害）。
  - 真·弹道（proj）仅保留给实弹/元素球/穿甲弹/榴弹等本就是弹道的塔。
- 验证：无头自动战铺设 15+ 种各原型塔 + L+L→LL 合成，全 5 波跑通、零运行时报错。

### M2（拖拽合成 + 放置系统改造）
- **网格自由放置**：取消固定塔位，除路径/国王/顶部 HUD 外任意处可建；不同塔占不同格数（机枪猪1格、激光猪2格、元素猪4格，见 tower_defs `cells`）。
- **格子悬停才显示**：鼠标所在格高亮（绿=可建/红=不可），不再常亮。
- **点击→三选一**：点击空地弹出三基础塔选项（名/价/格数）→ 选择放置；金币不足或放不下有提示。
- **拖拽合成**：按住塔拖到另一同级塔松手 → 走融合表合成（同级升级/二级配对出三级/三级封顶），拖拽时有连线与目标可合成高亮（绿/红）。
- 右键拆除（返还 60% 金币）保留；战斗中全程可放置/拆除/合成。
- 验证：无头自动战——网格放置 8 塔、B+B→BB 合成、全 5 波跑通，放置/合成/拆除/胜负路径无运行时报错。

### M1（数据化地基）

主题确立为「猪猪打狼」，完成 M1：数据化地基 + 30 塔确定性融合表 + 游戏流程阶段机 + 大肥猪国王胜负机制。

- 立项 → 改名 → 项目结构（scripts/ scenes/ assets/ data/）→ 第一个可玩切片。
- 主题：猪（塔）打狼（敌人）。塔=猪持武器，敌人=各种狼。
- **数据层**：
  - `data/tower_defs.gd`：30 塔目录 + 完整确定性融合表（3 基础 → 6 二级 → 21 三级，路径决定产物）。无头自检通过：27 配方、30 塔、3/6/21 分级、21 组二级配对全有产物、0 失败。
  - `data/enemy_defs.gd`：3 种狼（野狼/疾风狼/铁背狼，含护甲减伤）+ 双 Boss 占位（头狼/披着羊皮的狼，特殊机制留待 M3）。
  - `data/levels.gd`：关卡1 草原村道、关卡2 黑森林雪原（路径 + 备用分支 + 塔位 + 波次 + 国王位）。
- **流程与机制**：点击开始 → 10s 准备倒计时（布防）→ 战斗（按波次出怪）→ 结算；战斗中可放置/拆除（返还 60% 金币）；3 基础塔（激光穿透/元素范围+debuff/机枪单体）数据驱动；元素 debuff（减速/灼烧）。
- **大肥猪国王**：路径终点为国王，狼到终点转「围攻」持续攻击国王掉血，靠近的塔仍可击杀；`king_hp` 归零=失败，扛完全部波次并清场=胜利（取代抽象生命值）。
- **验证**：Godot 4.6 无头模式——零解析错误、可启动；自动战斗跑通全 5 波（含 Boss），国王存活，胜利判定正确，无运行时报错。
