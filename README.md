# Bulwark（代号）

塔防游戏项目。**前期主电脑端（PC）**，使用 **Godot 4** 开发。

> 当前状态：**仅立项**。这是一个可在 Godot 4 中打开的最小空工程骨架，尚未开始玩法开发或策划。

## 环境

- 引擎：Godot 4.3+（本机暂未安装，开发前需先安装 Godot 4）
- 主场景：`main.tscn`（占位场景）

## 运行

用 Godot 4 编辑器导入本目录（选中 `project.godot`），按 F5 运行。

## 目录

```
project.godot   引擎配置
main.tscn       主场景（占位）
icon.svg        应用图标
修改日志.md      变更总日志（顶部为"未发布"区）
```
